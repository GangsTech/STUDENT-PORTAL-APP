# Student Portal App

This project is a Student Portal Application designed to manage student information such as profiles, attendance, marks, and notifications. The app provides secure login functionality for students and administrators.

## Features
- Student login and profile
- View attendance and marks
- Admin panel for managing data
- Responsive UI design

## Technology Used
- Python backend (Django/Flask)
- HTML, CSS, JavaScript for frontend

Project initialization completed.
Development in progress.
